BUG  : reported bugs.
FIXME: things are broken
XXX  : warning about possible pitfalls or problematic code
NOTE : addition info, tips or idea.
TODO : more code to be added, usually when skipping something.

HACK : Temporary code to force inflexible functionality or workaround a known problem

OPTIMIZE: 可能影响性能的缓慢或效率低下的代码
